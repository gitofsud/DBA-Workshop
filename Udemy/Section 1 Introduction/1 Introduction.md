# Introduction

* Oracle DB consists of 2 major components
    - DB storage
    - Oracle Instance

* Type of DB Files
    - Data files
    - Control files
    - Redo Log files
    - Database Backups
    - Archived Redo Logs
    - Parameter file
    - Password file
    - Alert Log file
    - Trace Files

## DB Instance

- Set of memory structures and processes that managed DB files
- Created by Oracle everytime you start DB
- Reading from memory is lot faster than reading from disk
